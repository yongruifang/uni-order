这是一个使用 `hbuilderx` + `uni-app` + `vue3` + `tailwind` 构建的小程序模板。可以直接在 `hbuilderx` 中导入运行。

目前已经升级到了 v2 版本

## HbuilderX 智能提示工具

DCloud-HBuilderX团队提供了对应的插件，可以去

<https://ext.dcloud.net.cn/plugin?id=8560> 进行下载，即可产生智能提示。
